# NHC / Method NHC — фирменный стиль

## Когда применять

Любые экраны продукта, лендинги и маркетинговые блоки, которые должны **визуально и тонально совпадать** с публичными страницами [nhc.live](https://nhc.live) (например, страницы ретритов вида `/p/...`).

## Визуал

### Настроение

Спокойствие, премиальность, природа и ясность. Минимум шума, много воздуха, **атмосферные фото** (лес, пространство ретрита) с **тёмным оверлеем** под белый текст.

### Цвета (ориентиры)

| Роль | Назначение |
|------|------------|
| Текст на тёмном / фото | Белый `#FFFFFF` |
| Основной CTA («Забронировать» и аналоги) | Приглушённый сине‑зелёный / тёмный циан — на проде снять точный hex из DevTools (референс ~`#4D7C7A`, не копировать вслепую) |
| Чипы даты / локации на герое | Полупрозрачный тёмный фон (`bg-black/40`–`bg-black/50` или градиент) |
| Фон секций без фото | Нейтральные тёпло‑холодные тёмные или светлые блоки в духе сайта — не кислотные |

**Правило:** перед фиксацией токенов в коде — **сверить** актуальные значения с живой страницей nhc.live.

### Типографика

- Семейство: **современный геометрический sans-serif** (визуально близко к Montserrat / Inter / Gilroy). Подключать то же, что на целевом лендинге NHC, или ближайший аналог в проекте.
- Иерархия: крупный **жирный** заголовок → подзаголовок легче и меньше → body читабельный, без узких condensed для длинных текстов.

### UI‑формы

- **Сильное скругление:** капсулы для CTA и верхних чипов (`rounded-full` или очень большой `rounded-*`).
- Кнопка основного действия — **широкая плашка**, контраст к фону, текст светлый на акцентном фоне.

### Композиция

- Герой: **full-bleed** изображение, `object-cover`, центр по смыслу; контент по центру или по сетке как на референсе.
- Шапка: логотип слева, навигация справа (часто бургер на мобильных).
- Логотип NHC: **белый контур головы** с «топографическими» линиями внутри — не заменять случайной иконкой.

## Копирайт и структура

- Заголовки: **образ + польза** (тишина, трансформация, внутренняя работа), без агрессивных обещаний.
- Сразу после эмоции — **факты:** даты, город, формат (экспресс / полный ретрит).
- Баланс формулировок: **практика + «научность»** (Method NHC, нейропластичность, структура) — без фанатичного жаргона.
- Социальное доказательство: **имя, роль, цитата** в отдельных карточках или блоках.
- Повтор **одного главного CTA** (например «Забронировать») в ключевых местах скролла.
- Прозрачность про **оргсбор / донаты / что входит** — тон спокойный, уважительный.

## Реализация в коде (кратко)

- Фото героя + `linear-gradient` или полупрозрачный слой поверх для читаемости текста.
- Токены: вынести `--nhc-accent`, `--nhc-hero-overlay` после сверки с продом.
- Не вводить третий «брендовый» акцент без необходимости; **один** основной цвет кнопки + нейтрали.

## Чеклист перед сдачей

- [ ] CTA и чипы со скруглением в духе nhc.live
- [ ] Текст на фото читается на реальном изображении
- [ ] Тон текстов совпадает (спокойно, взросло, без крика)
- [ ] Акцентный цвет сверен с актуальной страницей NHC

## Дополнительно

При смене дизайна на nhc.live обновить таблицу цветов и примечание про шрифт в этом skill.

**Где Cursor видит skill:** список Agent Skills и автоподхват обычно идут из `~/.cursor/skills/<имя>/SKILL.md`. Копия в этом репозитории — для git; при изменениях синхронизируй с глобальной папкой или перезапусти Cursor после добавления файла.
